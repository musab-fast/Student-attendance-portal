# Color Theme Migration Guide

## Old Theme → New Theme Mapping

### Background Colors
- `#0A0F1F` → `#0D0D0D` (Main Background)
- `#11182B` → `#222222` (Cards/Surface)
- `#141414` → `#141414` (Sidebar - Keep)

### Text Colors
- `#E2E8F0` → `#F5F5F5` (Primary Text)
- `#94A3B8` → `#D1D5DB` (Secondary Text)
- `#64748B` → `#6B7280` (Muted Text)

### Border Colors
- `#1E293B` → `#2E2E2E` (Borders)
- `#334155` → `#2E2E2E` (Borders)

### Accent Colors (Blue)
- `#4F46E5` → `#1D4ED8` (Primary Blue)
- `#4338CA` → `#1E40AF` (Hover Blue)
- `#3B82F6` → `#1D4ED8` (Blue variant)

### Status Colors (Keep neutral greys or blue)
- `#10B981` (Green) → `#1D4ED8` (Blue for success)
- `#059669` (Dark Green) → `#1E40AF` (Dark Blue)
- `#EF4444` (Red) → `#6B7280` (Grey for danger/warning)
- `#DC2626` (Dark Red) → `#4B5563` (Dark Grey)
- `#F59E0B` (Orange) → `#6B7280` (Grey)
- `#D97706` (Dark Orange) → `#4B5563` (Dark Grey)

### Greys
- `#475569` → `#4B5563`
- `#374151` → `#374151` (Keep)
